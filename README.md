# DBX Auto-PDF Generator 🖨️🤖

> **Plugin para o ambiente DBX que automatiza a geração de PDFs a partir de desenhos CAD, mantendo a consistência visual e o layout padrão do ecossistema.**

---

## 📝 Sobre o Projeto

O **DBX Auto-PDF Generator** é uma ferramenta desenvolvida para otimizar o fluxo de trabalho de engenharia e design. Em vez de abrir manualmente cada arquivo CAD e exportar para PDF configurando layouts repetitivos, este plugin automatiza todo o processo diretamente na interface do DBX.

A ferramenta foi projetada seguindo rigorosamente os padrões visuais e de usabilidade do **layout DBX**, garantindo uma experiência integrada e fluida para o usuário.

### ✨ Principais Funcionalidades

* **Exportação em Lote:** Selecione múltiplos desenhos CAD e gere todos os PDFs de uma só vez.
* **Fidelidade ao Layout DBX:** Interface nativa e alinhada com o ecossistema que você já conhece.
* **Mapeamento Automático de Folhas:** Detecção automática do tamanho da folha (A4, A3, A2, A1, A0) e orientação (Retrato/Paisagem).
* **Padronização de Nomenclatura:** Os arquivos gerados são salvos automaticamente seguindo as regras de nomenclatura do projeto (Ex: `Codigo_Revisao_Nome.pdf`).
